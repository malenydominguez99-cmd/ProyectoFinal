# FanPage BTS

Esta es una página web creada con HTML y CSS que presenta información sobre mi banda favorita, **BTS**. Incluye biografía, integrantes, historia, premios, canciones más populares y datos sobre su fandom, ARMY.

## Características
- Estructura con HTML semántico (`header`, `main`, `section`, `footer`)
- Uso de títulos (`h1`, `h2`, `h3`), párrafos, listas, enlaces e imágenes
- Diseño responsive y visual atractivo con CSS
- Información completa sobre cada integrante y su carrera
- Navegación fácil con enlaces internos y externos
